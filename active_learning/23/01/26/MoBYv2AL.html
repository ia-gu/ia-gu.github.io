MoBYv2AL: Self-supervised Active Learning for Image Classification

https://arxiv.org/abs/2301.01531

Jan 26, 2023

Active Learning

BMVC (2022)

1. どんなもの??何ができる??

  • Swin向けに提案されたSSL手法MoBYを改良し，DALと組み合わせたフレームワークMoBYv2ALを提案
  • CIFAR，SVHN，不均衡CIFARなどで有効性を確認

2. どうやって??

  • MoBYのEncoderをSwin TransformerからCNNに変更
  • 講師あり学習の中でMoBYの学習とクラス分類を行う
    • MoBYのLoss|query，keyによるInfoNCEの和

  • 学習全体のLoss(λ=0.5)

  • ↓フレームワーク全体

    • クエリ戦略にはCore-setを使用(任意の戦略を使用可能)

3. 結果は??

  • CIFAR10，CIFAR100 比較結果

    • VGG16にしてはかなり精度が高い
  • SVHN，FashionMNIST 比較結果

  • 不均衡CIFAR10 比較結果

4. 課題は??

  • 既存手法と既存手法を組み合わせただけで，特にひねりもない
