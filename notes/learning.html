Machine Learning

どんなもの??

機械学習の学習方法をまとめたものです．(制作時｜B4)

画像はココから拝借しました

教師あり学習｜Supervised Learning

モデルの出力と教師ラベルの誤差を小さくするようにモデルのパラメータを最適化する学習方法

弱教師あり学習｜Weakly supervised Learning

目的タスクとは別のタスクに向けて用意された教師ラベルを用いる学習方法

教師なし学習｜Unsupvervised Learning

学習の際，ラベルを使用しない学習方法

例｜クラスタリング，SVM，自己教師あり学習(後述)

半教師あり学習｜Semi-Supervised Learning

学習に使用するデータの一部に教師ラベルが付与されており，教師付きデータと教師なしデータを同時に使用する学習方法

細かい解説は別途追加予定

自己教師あり学習｜Self-Supervised Learning

学習データの教師ラベルをモデルが機械的に生成し，その教師ラベルを使用する学習方法(前述の教師なし学習の一種)

細かい解説は別途追加予定

能動学習｜Active Learning

モデルが自身にとって有益と判断したデータにのみ教師ラベルを付与することで，効率的に学習を行う問題設定

細かい解説は別途追加予定

few-shot Learning

学習データが1クラスN枚しか存在しない教師あり学習の問題設定

Nは1~5と，ごく少数な場合がほとんど

zero-shot Learning

対象タスクに対する学習データが1枚も存在しない教師あり学習の問題設定

類似したドメインを用いて学習を行い，類推の要領でタスクに取り組むため，教師なし学習とは問題設定が異なる点に注意

例｜馬，縞模様の学習をしたモデルは，初めてシマウマを見たときシマウマと認識できる(可能性が高い)

距離学習｜Metric Learning

特徴空間上で特徴量同士の距離を処理しやすいように学習する手法

Contrastive Loss，Center Lossなどが代表例であり，few-shot Learningに用いられる

Meta Learning

学習の行い方やハイパーパラメータといったメタ的な情報を学習する手法

例｜MAML(詳細は別途追加予定)

転移学習｜Transfer Learning

別のタスクやドメイン(ソースドメイン)で学習済みのモデルを一部利用してメインのタスク(ターゲットドメイン)の学習を行う手法

抜き出したモデルパラメータは更新しないため，学習にかかる時間が短縮可能

参考｜パラメータ更新をする手法としてファインチューニングが存在

継続学習｜Continuous Learning

あるタスクに特化したモデルが別のタスクで学習しなおした時，どちらのタスクにも対応できるように学習する問題設定

強化学習｜Reinforcement Learning

エージェントの行動に報酬を与え，その報酬が最大になるように学習する手法

盤面数(パラメータ)が膨大になるゲームなどで利用

模倣学習｜Immitation Learning

ヒトの行動をモデルが模倣するように学習する問題設定

ロボット操作や自動運転といった処理で利用

ドメイン適応｜Domain Adaptation

学習とテストで使用するデータの間に生じる差を小さくするための手法

テストデータのラベルの有無によって，教師なし/半教師あり/教師ありドメイン適応に分類
